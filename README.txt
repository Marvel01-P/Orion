Orion Bot (MetaSearch Deluxe) scaffold created.

How to build:
1. Ensure JDK 11+, Android SDK & platform tools installed.
2. From this folder run:
   flutter pub get
   flutter build apk --release

If you don't have the Android SDK, open this folder in Android Studio and follow prompts.

APK (if build succeeds) will be at:
build/app/outputs/flutter-apk/app-release.apk
